Code Key

Use these values for the numeric fields in the form and CSV import.

source_code

  • 1 = website form
  • 2 = admin add
  • 3 = csv import
  • 4 = member signup
  • 5 = api

preferred_contact_code

  • 1 = email
  • 2 = sms
  • 3 = either
  • 4 = none

consent_source_code

  • 1 = website checkbox
  • 2 = admin recorded
  • 3 = imported
  • 4 = api
